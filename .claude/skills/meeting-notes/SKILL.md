# meeting-notes（會議紀錄／開會導向）

給**內部討論會議**（多人、有決議與待辦）用，取代演講導向的 summarize。
輸入 `錄音/transcript.md`（＋`transcript.speakers.md` 講者分離）→ 產 `summary.md`：
結論/決議 → 逐議題討論 → **行動項表格（負責人/期限/狀態）** → 未決/待追蹤 → 風險/爭議 → 存疑。

由 `meeting-notes-agent` 執行。行動項的負責人/期限只從逐字稿抽取，抽不到標「（未指定）」，不編造。
建議搭配講者分離（`錄音/.diarize`）把主張/拍板歸屬到人。
